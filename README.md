# trapstreet-mvp

> **Find the fakes. Run real evals. Share the results.**
> [trapstreet.run](https://trapstreet.run)

The community evaluation harness for AI workflows, agents, and skills.

> Runner picks a Mission under a Track, opens a Run, runs it, server scores it,
> the result lands on the Leaderboard. Everyone discusses on Threads.

This repo is the **single truth source** for trapstreet v0 implementation.
Branding, landing site, and earlier design discussions stay in the old
`AntiNoise-ai/trapstreet` repo.
